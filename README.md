## 🚀 Overview

Fashion Store Admin Dashboard is a modern retail management system built for clothing shops, garment stores, boutiques, and fashion businesses.

It helps shop owners and staff efficiently manage inventory, track sales, monitor orders, analyze business performance, and streamline daily operations through a clean and intuitive dashboard.
